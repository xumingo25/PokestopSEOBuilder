# Pokestop SEO Builder

MVP para transformar exportaciones de productos de Tienda Nube en archivos enriquecidos con columnas SEO y contenido personalizado.

## Stack

- HTML
- CSS
- TypeScript
- Vite

Sin frameworks de UI.

## Scripts

```bash
npm install
npm run dev
npm run build
```

## Sprint 1

El primer sprint entrega una aplicacion local capaz de:

- Cargar archivos CSV o TSV exportados desde Tienda Nube.
- Detectar columnas comunes de producto.
- Leer descripciones HTML.
- Sugerir titulo SEO, meta descripcion, slug y descripcion limpia.
- Previsualizar resultados antes de exportar.
- Descargar un CSV enriquecido.
